Tentang Emotify

Aplikasi deteksi ekspresi wajah real-time yang berjalan langsung di browser menggunakan Face-API.js.

Apa itu Emotify?

Emotify adalah aplikasi berbasis web yang mampu mendeteksi ekspresi wajah manusia secara real-time menggunakan webcam. Aplikasi ini berjalan sepenuhnya di sisi pengguna (client-side) sehingga:

  • Tidak mengirim data wajah ke server
  • Aman dan menjaga privasi
  • Bekerja cepat karena proses dilakukan di browser
Emotify memanfaatkan teknologi Face-API.js (TensorFlow.js), yaitu model machine learning ringan yang mampu mengenali wajah, titik-titik landmark, dan ekspresi dasar manusia.

Deteksi Wajah

Menggunakan Tiny Face Detector – model ringan berbasis TensorFlow.js yang dirancang untuk performa cepat dan efisien langsung di browser.

Landmark Wajah

Emotify mendeteksi 68 titik landmark wajah untuk memahami struktur wajah secara presisi.

Pengenalan Ekspresi

Menganalisis 7 ekspresi dasar termasuk: netral, senang, sedih, marah, takut, terkejut, dan jijik.

Fitur Utama

Real-Time Detection

Analisis ekspresi wajah langsung dari kamera Anda dalam hitungan milidetik.

Privasi Terjaga

Semua pemrosesan dilakukan di perangkat Anda. Tidak ada data wajah yang dikirim ke server manapun.

Berbasis Browser

Tidak memerlukan instalasi apa pun – cukup gunakan browser modern seperti Chrome atau Edge.
